# C-term3D_Website

A website implementation of my C program [C-term3D](https://github.com/ashleythatcatgirl/C-term3D)

# Description

## Overview

The basic idea is to implement my C rendering program for 3D scenes with an api from [Poly Haven](https://polyhaven.com) for downloading those models, and immediately viewing them in your browser

## Details

### Frontend

Made with react (typescript), and Tailwind for css since im not a graphics designer and I can't be trusted with designing my own css
The main part is a viewport of the C program with basic controls for camera and such

### Backend

#### Api (Poly Haven)
- Search field and filtering based on metadata
- Saving favorite models
